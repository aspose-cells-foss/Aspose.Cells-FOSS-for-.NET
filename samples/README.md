# Samples

This directory contains runnable console projects that demonstrate implemented features in `Aspose.Cells_FOSS`.

## How to run

Run any sample from the repository root:

```bash
dotnet run --project samples\<SampleProjectFolder>\<SampleProjectName>.csproj
```

Example:

```bash
dotnet run --project samples\Aspose.Cells_FOSS.Samples.Basic\Aspose.Cells_FOSS.Samples.Basic.csproj
```
