# Samples

This folder contains small console programs that exercise implemented features in `Aspose.Cells_FOSS`.

- `Aspose.Cells_FOSS.Samples.Basic`: basic cell values, formulas, and file round-trip.
- `Aspose.Cells_FOSS.Samples.Loading`: load with repair-oriented `LoadOptions`.
- `Aspose.Cells_FOSS.Samples.Styles`: cell style creation and inspection.
- `Aspose.Cells_FOSS.Samples.WorksheetSettings`: worksheet visibility, view settings, protection, rows, columns, and merges.
- `Aspose.Cells_FOSS.Samples.Validations`: list, decimal, and custom data validations.
- `Aspose.Cells_FOSS.Samples.ConditionalFormatting`: basic and advanced conditional-formatting rules.
- `Aspose.Cells_FOSS.Samples.HyperlinksAndNames`: external and internal hyperlinks plus workbook defined names.
- `Aspose.Cells_FOSS.Samples.PageSetup`: print settings, headers and footers, margins, and page breaks.
- `Aspose.Cells_FOSS.Samples.Comments`: cell comments with authors, notes, visibility, and sizing.
- `Aspose.Cells_FOSS.Samples.Charts`: column and line charts anchored to data ranges.
- `Aspose.Cells_FOSS.Samples.ListObjects`: Excel tables with headers, totals, styles, and column management.
- `Aspose.Cells_FOSS.Samples.Pictures`: embedding images in worksheets with anchor positioning.
- `Aspose.Cells_FOSS.Samples.DocumentProperties`: workbook metadata including title, author, keywords, and custom properties.
- `Aspose.Cells_FOSS.Samples.Shapes`: drawing objects including rectangles, ellipses, triangles, arrows, stars, and other auto shapes.

Run a sample with `dotnet run --project <sample-csproj>`.
